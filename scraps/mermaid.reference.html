commited date: 2025-11-17

#Markdown

By specifying mermaid as the language for a code block, you can use Mermaid diagrams.

Example

In:


```mermaid
graph LR
    A --- B
    B-->C[fa:fa-ban forbidden]
    B-->D(fa:fa-spinner);
```

Out:

graph LR
    A --- B
    B-->C[fa:fa-ban forbidden]
    B-->D(fa:fa-spinner);
