#[[Emit/Static Site]]

The site adapts to the OS color scheme by default. Override the
`color_scheme` field of `[ssg]` in [[Reference/Configuration]]:

| Value | Behavior |
|---|---|
| `os_setting` (default) | Follow the user's OS preference |
| `only_light` | Force light |
| `only_dark` | Force dark |
