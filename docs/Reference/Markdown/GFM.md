#[[Notation/Markdown]]

GitHub-flavored Markdown extensions are supported: tables, task lists,
strikethrough, and autolinks.

```markdown
| Column | Column |
|---|---|
| cell | cell |

- [x] done
- [ ] open
- [-] deferred

~~strikethrough~~
```

GFM task list items are aggregated wiki-wide by `scraps todo` (see
[[Reference/CLI Overview]]).
